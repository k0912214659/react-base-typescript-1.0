### Dev Server Guide

1. Clone the project from [react-base-typescript-1.0](https://github.com/k0912214659/react-base-typescript-1.0.git).
2. Move the root path in project folder.
3. Run `npm i` or `npm install` to install node_modules.
4. The default server is on `localhost:3000`, please check you don't have any server on it.
5. Run `npm run start` to start dev server.

Steps

```git bash
git clone https://github.com/k0912214659/react-base-typescript-1.0.git
cd react-base-typescript-1.0
npm install && npm audit fix
npm run start
```
